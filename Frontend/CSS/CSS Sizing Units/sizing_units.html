px
pixel = 196th of an inch. Select according to your screen resolution such as 1920*1080
vh
Viewport - Height, used for responsive layout. Suppose, you have 2k screen and others have 1k screen. The sizes are adjusted according to screen resolution to avoid horizontal scroll bar.
Use Toggle Device Emulation to check layouts in different devices. 
            1vh = 1% of the viewport height

            So if the browser window is 1200px tall, then:

            1vh = 12px
            100vh = 1200px
em
The child paragraph in a container will inherit font-size(css property) as it is from container(parent) if font-size: em is not applied in child. But, if em > 1 is applied in child font-size, then its font size increases relative to that parent's font-size. 
                1em = current font-size.
                If a parent element has font-size: 16px, then: 
                1em = 16px
                2em = 32px
                0.5em = 8px
                Child element  will have size relative to parent.
em is used for padding, font-size, height, width and margin
